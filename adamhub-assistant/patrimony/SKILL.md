# AdamHUB Patrimony Skill

Use for net-worth review, account tracking, and savings goals.

## Actions

- `patrimony.overview`
- `patrimony.list_accounts`
- `patrimony.add_account`
- `patrimony.update_account`
- `patrimony.delete_account`
- `patrimony.list_goals`
- `patrimony.add_goal`
- `patrimony.update_goal`
- `patrimony.delete_goal`

## Rules

- Review `patrimony.overview` before proposing changes.
- Use live account balances when a savings goal is linked to an account.
- Confirm deletes explicitly.
